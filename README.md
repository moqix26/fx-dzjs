## 功能特点
- **从零讲起**：每章含前置知识点（欧姆定律、KVL、分压公式、PN 结等），适合基础薄弱的同学
- **逐题详解**：例题与习题均附完整解题步骤、公式推导与思路说明
- **生活化比喻**：用通俗类比帮助理解二极管、稳压管、晶体管等抽象概念
- **配图对照**：电路原题截图存放于 `assets/`，便于对照阅读
- **侧边栏导航**：按章节 / 题号快速跳转，支持移动端折叠菜单
- **考前工具**：方法论总结 + 分章易错点 + 速查表，适合考前快速回顾

## 本地运行
本项目为纯静态网页，无需 Node.js 或构建工具。
```bash
# 克隆仓库
git clone https://github.com/moqix26/fx-dzjs.git
cd fx-dzjs
# 启动本地服务器（任选其一）
python -m http.server 8080
# 或
npx serve .
浏览器访问 http://localhost:8080 即可。

使用建议
基础薄弱：先读第 1 章「前置知识点」，掌握欧姆定律、KVL、分压公式
系统复习：按章节顺序阅读例题解析，再独立完成习题并对照答案
考前冲刺：重点看「解题方法论总结」和「易错点清单」
临考速查：使用「考试速查表」快速回忆公式、编码规则与芯片功能
